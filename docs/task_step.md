# Overview

- 分析プロジェクトは以下に示す5段階で構成される

# Theme

- 電力変圧器の油温（Oil Temperature, OT）を予測するために、ETT-smallデータセット(`ETTh1.csv`, `ETTh2.csv`, `ETTm1.csv`, `ETTm2.csv`)を用いて、時系列データ解析を行う。

---

## 1. データの理解と前処理 (Data Understanding & Preprocessing Phase)

### 1.1 データ構造の把握
- データセットのカラム構造の確認（date, HUFL, HULL, MUFL, MULL, LUFL, LULL, OT）
- 各データセットのレコード数・期間の確認
- データ型の確認と変換（日時型への変換等）

### 1.2 データ品質の検証
- 欠損値の検出と対処方針の決定
- 異常値・外れ値の検出（IQR法、Zスコア等）
- 重複レコードの確認

### 1.3 探索的データ分析 (EDA)
- 記述統計量の算出（平均、中央値、分散、標準偏差、四分位数）
- 時系列プロット：OT（Oil Temperature）の時間変化を可視化
- 各特徴量（HUFL〜LULL）の分布確認
- 特徴量間の相関分析（相関行列、ヒートマップ）

### 1.4 時系列特性の分析
- トレンド成分の確認
- 季節性・周期性の分析（日次、週次、年次）
- 自己相関（ACF）・偏自己相関（PACF）の分析
- 定常性の検定（ADF検定、KPSS検定）

---

## 2. モデル選定 (Model Selection Phase)

### 2.1 候補モデルの検討
- 統計的時系列モデル
  - ARIMA / SARIMA
  - VAR（Vector Autoregression）
  - 指数平滑法（Exponential Smoothing）
- 機械学習モデル
  - 線形回帰（ベースライン）
  - Random Forest / Gradient Boosting
  - LSTM / GRU（深層学習）
- ハイブリッドモデル
  - Prophet
  - Informer（データセット提供元のモデル）

### 2.2 モデル選定基準の定義
- 予測精度（MAE, RMSE, MAPE等）
- 計算コスト・実行時間
- 解釈可能性
- 長期予測への適用可能性

### 2.3 ベースラインモデルの決定
- 比較基準となるシンプルなモデルの選定
- 予測ホライズン（短期/中期/長期）の定義

---

## 3. 実験設計 (Design Phase)

### 3.1 データ分割設計
- 訓練データ・検証データ・テストデータの分割方法
- 時系列の順序を保持した分割（データリーケージの防止）
- 分割比率の決定

### 3.2 検証手法の設計
- 時系列クロスバリデーションの設計
- ウォークフォワード検証（拡張ウィンドウ / スライディングウィンドウ）
- 予測ホライズンごとの検証設計

### 3.3 評価指標の定義
- 予測精度指標（MAE, RMSE, MAPE, R²）
- ベースラインモデルとの比較基準
- 統計的有意性検定の方法

---

## 4. 検証環境の設定 (Setup Phase)

### 4.1 開発環境の構築
- Python環境のセットアップ（venv / conda）
- 必要ライブラリのインストール（pandas, numpy, scikit-learn, statsmodels等）
- Jupyter Notebook / Lab の設定

### 4.2 データパイプラインの構築
- データ読み込み・前処理の自動化
- 特徴量エンジニアリングパイプライン
- モデル学習・評価パイプライン

### 4.3 再現性の確保
- 乱数シードの固定
- バージョン管理（Git）
- 実験管理（MLflow等の検討）

### 4.4 コード構成の設計
- ディレクトリ構造の決定
- モジュール分割方針
- 設定ファイル（config）の設計

---

## 5. 実装 (Implement Phase)

### 5.1 データ前処理の実装
- データ読み込み・クリーニング処理
- 特徴量エンジニアリング
- データ分割（train/val/test）

### 5.2 ベースラインモデルの実装
- 線形回帰モデルの構築
  - 6つの特徴量（HUFL, HULL, MUFL, MULL, LUFL, LULL）を用いてOTを予測
- ベースライン精度の確認

### 5.3 時系列モデルの実装
- 選定したモデルの実装
- ハイパーパラメータチューニング
- モデルの学習と予測

### 5.4 モデル評価
- 評価指標の算出
- ベースラインモデルとの比較
- 予測誤差の分析

### 5.5 結果の可視化・レポート作成
- 予測結果の可視化
- モデル比較表の作成
- 考察とまとめ
